# Changelog

All notable changes to OpenClaw Twins will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitOps architecture with GitHub Actions workflows
- Docker containerization support
- Kubernetes deployment manifests
- Canary deployment strategy
- Automated backup and rollback workflows

## [3.0.0] - 2026-03-20

### Added
- OpenClaw Twins rebranding from OpenClaw Agent OS
- Twin-Agent architecture: Overseer + Architect
- Real-time system monitoring dashboard
- Self-evolving optimization proposal system
- Budget governance with dynamic quotas
- Staff view for Agent session monitoring
- Proposal management with approval workflow
- SSE-based real-time data streaming
- 30-second snapshot cache with background refresh

### Changed
- Renamed all system references to OpenClaw Twins
- Updated documentation and branding

### Fixed
- Frontend API path resolution
- Password hashing with bcrypt compatibility
- Duplicate route registrations
- Tool call counting for OpenClaw JSONL format

## [2.0.0] - 2026-03-15

### Added
- Basic data collection infrastructure
- Simple monitoring reports
- Session analytics

## [1.0.0] - 2026-03-10

### Added
- Project initialization
- Concept validation
- Basic agent monitoring prototype
